Instagram Export Guide

인스타그램 내보내기 ZIP 받는 방법

ISeeSocial에서 인스타 언팔로워 후보와 맞팔 관계를 확인하려면 Instagram Accounts Center에서 JSON 형식의 내보내기 파일을 받아야 합니다. 이 과정에는 Instagram 로그인 정보나 비밀번호를 ISeeSocial에 입력하지 않습니다.

필수 설정 요약

  • 형식: JSON
  • 기간: 전체 기간
  • 포함 항목: 팔로워 및 팔로잉 또는 Connections 관련 항목
  • 다운로드 후 받은 ZIP을 압축 해제하지 말고 그대로 업로드 권장

다운로드 순서

  1. Instagram 앱 또는 웹에서 Accounts Center로 이동합니다.
  2. 내 정보 및 권한 메뉴에서 내 정보 다운로드를 선택합니다.
  3. 기기로 내보내기를 선택합니다.
  4. 정보 맞춤 설정에서 팔로워 및 팔로잉 관련 항목을 포함합니다.
  5. 기간은 전체 기간으로 설정합니다.
  6. 형식은 반드시 JSON으로 설정합니다.
  7. Instagram이 준비한 ZIP 파일을 다운로드합니다.

자주 틀리는 부분

HTML로 받은 경우

HTML 내보내기는 사람이 읽기에는 편하지만 ISeeSocial 분석에는 맞지 않습니다. JSON 형식으로 다시 요청해야 합니다.

기간이 일부 기간인 경우

일부 기간만 받으면 예전 팔로우/팔로워 관계가 빠져 언팔로워 후보 수치가 이상해질 수 있습니다.

followers/following 파일이 없는 경우

ZIP 내부에 `followers_1.json`, `following.json` 또는 유사한 관계 파일이 없다면 분석할 수 없습니다. 내보내기 항목을 다시 확인하세요.

분석 결과가 이상해 보일 때

결과는 Instagram 내보내기 파일을 생성한 시점의 스냅샷입니다. 현재 Instagram 앱 화면과 차이가 날 수 있고, 계정명 변경이나 삭제 계정 때문에 체감과 다르게 보일 수도 있습니다. ISeeSocial의 사용 파일 진단에서 어떤 JSON 파일을 읽었는지 먼저 확인하세요.

개인정보 기준

ISeeSocial은 ZIP, 파일명, 사용자명, 결과 목록을 서버로 전송하지 않습니다. 자세한 내용은 개인정보 안내에서 확인할 수 있습니다.
